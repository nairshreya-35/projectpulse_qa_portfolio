## Quick Overview

# ProjectPulse SaaS Testing Case Study

## Overview

ProjectPulse is a fictional SaaS project management platform created to demonstrate end-to-end Quality Assurance practices.

This repository showcases the testing activities typically performed by a QA Engineer during the Software Testing Life Cycle (STLC), including:

* Requirement Analysis
* Test Planning
* Test Case Design
* Test Execution
* Defect Reporting
* Exploratory Testing
* API Testing
* Traceability Management

This project is intended as a professional QA portfolio and does not represent a production system.

---

ProjectPulse is a cloud-based project management application that helps teams:
